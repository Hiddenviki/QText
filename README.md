### Features

- Tesseract 4.0 —ÅLSTM data;
- Modern UI/UX Material Design Interface
- Support Latin (English) and Cyrillic (Russian and similar languages)
- Kotlin asynchronous methods (coroutines) are used to get the result
- Fast Recognition
- Support Night Mode
